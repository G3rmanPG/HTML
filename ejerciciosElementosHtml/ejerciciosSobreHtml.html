Historia del Nacimiento de la Web

Escriba un documento HTML con un breve historia de HTML. El documento tendrá un encabezado principal y cuatro secciones

Todas las apariciones del acrónimo HTML aparecerán en negrita y el nombre de Tim Berners-Lee siempre aparecerá en cursiva

La frustración en el CERN

Corre el año 1989. En los fríos pasillos del CERN (La Organización Europea para la Investigación Nuclear) en Suiza, un joven físico e informático británico llamado Tim Berners-Lee caminaba frustrado.

El problema de Tim no era la física de partículas, sino la información. El CERN estaba lleno de científicos brillantes con computadoras diferentes, sistemas diferentes y formatos de documentos incompatibles. Si Tim quería leer el informe de un colega, a menudo tenía que caminar físicamente a otro edificio o aprender a usar un programa completamente nuevo. Era una Torre de Babel digital.

La Visión

Tim se sentó frente a su estación de trabajo y tuvo una visión. Imaginó un mundo donde cualquier documento pudiera conectarse con otro, sin importar en qué computadora estuviera. Llamó a este proyecto "World Wide Web". Pero para que esta telaraña mundial funcionara, Tim necesitaba un lenguaje común.

Un idioma que todas las computadoras pudieran hablar y entender. Tim recordó un viejo sistema de formateo llamado SGML, que era poderoso pero terriblemente complejo. Él necesitaba algo más sencillo, algo rápido, algo que pudiera viajar por los cables de internet sin pesar demasiado. Así que, simplificando al máximo, Tim comenzó a escribir las reglas de lo que hoy llamamos HTML (Hipertext Markup Language).

Creando las etiquetas

—Debe ser simple —pensó Tim mientras tecleaba—. Si el HTML es difícil de aprender, nadie lo usará.

Tim definió unas pocas "etiquetas" básicas. Decidió que si querías un título, usarías <h1> en HTML. Si querías un párrafo, usarías <p>. Pero la etiqueta más revolucionaria que creó para el HTML fue la etiqueta de anclaje <a>.

Con esa pequeña etiqueta de HTML, Tim logró lo imposible: el hipervínculo. Ahora, una palabra en un documento podía "teletransportar" al usuario a otro documento completamente diferente.

El legado

En agosto de 1991, Tim publicó la primera página web de la historia. No tenía colores, ni vídeos, ni logotipos. Era, sencillamente, texto puro estructurado con HTML. En esa página, Tim explicaba qué era la web y cómo usarla. Cualquier persona que accediera a esa página, si hacía clic en "Ver código fuente", veía algo sorprendente: el HTML era legible por humanos. No eran ceros y unos incomprensibles. Eran palabras en inglés entre corchetes angulares.

Tim tomó una decisión histórica: no patentó el HTML. No cobró regalías por su uso. Lo regaló al mundo. Sabía que si el HTML pertenecía a una sola empresa, la Web moriría antes de nacer. Gracias a esa generosidad y a la simplicidad del HTML, estudiantes, aficionados y científicos de todo el mundo empezaron a crear sus propias páginas. El HTML se convirtió en el ladrillo con el que se construyó la era digital.

Hoy, más de 30 años después, cada vez que abres una red social, lees una noticia o compras algo en línea, el navegador sigue haciendo exactamente lo mismo que hizo la computadora NeXT de Tim aquel día: leer el código HTML y dibujar el mundo ante tus ojos.

Características Principales HTML

HTML 1.0 (1993):
Versión original creada por Tim Berners-Lee. Era extremadamente básica, diseñada principalmente para compartir información científica.
Soportaba texto, enlaces y una estructura muy simple.
HTML 2.0 (1995):
Primer estándar oficial creado por la IETF (Internet Engineering Task Force). Introdujo elementos fundamentales que aún usamos hoy como los formularios básicos para introducir datos.
HTML 3.2 (1997):
Creado por el W3C (World Wide Web Consortium). Añadió soporte para tablas complejas, applets de Java y opciones para alinear texto alrededor de imágenes.
Fue una época donde se empezó a usar HTML para diseñar visualmente, lo cual era un error.
HTML 4.01 (1999):
Un estándar crucial que dominó internet durante muchos años. Su mayor logro fue empujar a los desarrolladores a separar la estructura del diseño visual, recomendando usar CSS para los estilos y dejando a HTML solo para el contenido
XHTML 1.0 (2000):
Fue una reescritura de HTML 4.01 pero basándose en las reglas estrictas de XML. Obligaba a los programadores a ser muy limpios y disciplinados
(por ejemplo, todas las etiquetas debían cerrarse obligatoriamente y escribirse en minúsculas).
HTML5 (2014 - Actualidad):
La gran revolución web moderna. Introdujo etiquetas semánticas (header, footer, article, nav), eliminó la necesidad de usar plugins de terceros como Adobe
Flash gracias a sus etiquetas nativas de audio y video, y añadió capacidades avanzadas como dibujar gráficos (Canvas) y guardar datos en el navegador (Local Storage).

Características HTML:5

  1. Es un lenguaje de marcado, no de programación
  2. Se lee e interpreta, no se compila
  3. Estructura de árbol (DOM)
  4. Universal y multiplataforma
  5. Tolerante a errores
  6. Altamente colaborativo
  7. Enfoque en la Accesibilidad (Semántica)

El creador de la Web tenía una visión muy clara cuando dijo que la web es para todos, un principio que cambió el mundo para siempre.

El poder de la Web está en su universalidad. El acceso por parte de cualquier persona, independientemente de su discapacidad, es un aspecto esencial.

Creando las etiquetas

—Debe ser simple —pensó Tim mientras tecleaba—. Si el HTML es difícil de aprender, nadie lo usará.

Tim definió unas pocas "etiquetas" básicas. Decidió que si querías un título, usarías <h1> en HTML. Si querías un párrafo, usarías <p>. Pero la etiqueta más revolucionaria que creó para el HTML fue la etiqueta de anclaje <a>.

Con esa pequeña etiqueta de HTML, Tim logró lo imposible: el hipervínculo. Ahora, una palabra en un documento podía "teletransportar" al usuario a otro documento completamente diferente.

El legado

En agosto de 1991, Tim publicó la primera página web de la historia. No tenía colores, ni vídeos, ni logotipos. Era, sencillamente, texto puro estructurado con HTML. En esa página, Tim explicaba qué era la web y cómo usarla. Cualquier persona que accediera a esa página, si hacía clic en "Ver código fuente", veía algo sorprendente: el HTML era legible por humanos. No eran ceros y unos incomprensibles. Eran palabras en inglés entre corchetes angulares.

Tim tomó una decisión histórica: no patentó el HTML. No cobró regalías por su uso. Lo regaló al mundo. Sabía que si el HTML pertenecía a una sola empresa, la Web moriría antes de nacer. Gracias a esa generosidad y a la simplicidad del HTML, estudiantes, aficionados y científicos de todo el mundo empezaron a crear sus propias páginas. El HTML se convirtió en el ladrillo con el que se construyó la era digital.

Hoy, más de 30 años después, cada vez que abres una red social, lees una noticia o compras algo en línea, el navegador sigue haciendo exactamente lo mismo que hizo la computadora NeXT de Tim aquel día: leer el código HTML y dibujar el mundo ante tus ojos.
